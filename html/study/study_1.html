CSS flexible Box

flex는 요소의 크기가 불분명하거나 동적인 경우에도 각 요소를 정렬할 수 있는 효율적인 방법을 제공한다.
플렉스는 2가지의 개념으로 나누어 지며, 첫번째는 container, 두번째는 items 이다.
container는 items를 감싸는 부모요소이며, 각 items를 정렬하기 위해서는 container가 필요하다.
주의할부분은 container와 item에 적용하는 속성이 구분되어 있다는 점 이다.

[container] : display / flex-flow / justify-content
[items] : order / flex /align-self

등을 사용할 수 있다.

[Display] - CONTAINER 속성

display 속성으로 flex container를 정의할 수 있다.
flex - block 특성의 flex container를 설정,
inline-flex - inline 특성의 flex container를 설정
*items에는 영향을 주지않는다. display 속성은container에만 인라인,블록 속성을 주는것임.*

@keyframes 애니메이션을 지속하는 시간에 동작을 정해준다..

window.innerWidth & window.innerHeight

자바스크립트에서 window 객체는 다양한 역할을 한다. 이 중 window.innerWidth와 window.innerHeight는 현재 창의 내부 넓이와 높이를 픽셀단위로 알 수 있다. 

    function updateScreenSize() {
      // 변수를 선언한다.
      var screenWidth = document.getElementById("screenWidth");
      var screenHeight = document.getElementById("screenHeight");

      // 현재 창의 넓이와 높이를 가져와서 해당 변수에 넣었다.
      screenWidthElement.textContent = screenWidth;
      screenHeightElement.textContent = screenHeight;
    }
    // 초기에 화면 크기 정보 업데이트
    updateScreenSize();
    // 브라우저 크기가 변경될 때마다 화면 크기 정보 업데이트
    window.addEventListener("resize", updateScreenSize);

< 실행 결과 >
화면 너비: px, 화면 높이: px

또한, 화면크기를 얻을때에는 초기 화면 크기정보를 업데이트 하기위해 updateScreenSize(); 함수를 한번 더 호출하고, 사용자에 의해 브라우저 크기가 변경 될 수 있으므로 변경될때마다 화면의크기를 업데이트 해줄 수 있는 코드를 덧붙여줘야한다. 

          // 초기에 화면 크기 정보 업데이트
          updateScreenSize();

          // 브라우저 크기가 변경될 때마다 화면 크기 정보 업데이트
          window.addEventListener("resize", updateScreenSize);
